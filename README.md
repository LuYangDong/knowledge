<h6>路由有什么作用呢？</h6>
<p>路由其实可以理解成 网站上不同的网页地址， 如果不是SPA的网站， 浏览器的前进后退，链接到了一个新的页面，整个页面重新刷新； 但如果是SPA网站，因为整个页面是不需要全部刷新的，网站的页面也停留在当前页面，那么怎么解决浏览器的前进后退问题呢， 怎么解决需要定位到特定网页地址的问题呢，所以SPA就引入了路由系统。</p>
